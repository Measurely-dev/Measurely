## Website Tracking Template

This template allows you to track page views and unique visitors automatically by adding a script tag to your website.

### How to Use

1. Copy the following script tag and add it to the `<head>` of your HTML file:

   ```html
   <!--Replace "YOUR_MEASURELY_API_KEY" with your actual API key.-->

   <script
     async
     src="https://cdn.measurely.dev/track.js"
     data-api-key="YOUR_MEASURELY_API_KEY"
   />
   ```

<Callout type="warning">
  {" "}
  The script will only collect data if the correct API key is provided. Ensure
  you are using a valid key.{" "}
</Callout>
