# Blocks

Blocks allow you to create customizable data visualizations blocks for your metrics. By defining blocks, you can display data in different types of charts, making it easier to interpret and analyze trends. For example, you can create a block to show revenue growth as a **area chart** or user distribution as a **bar chart**.

## How Blocks Work

Blocks enable you to:

- **Customize Visualizations:** Choose from different chart types (e.g., line, bar, pie) to display your data.
- **Highlight Key Insights:** Focus on specific trends and comparisons within your metrics.
- **Compare Data:** Use multiple blocks to view side-by-side metric performance.

## Creating and Using Blocks

<Callout type="warning">
  Ensure your blocks are properly configured to display accurate and meaningful
  data.
</Callout>

<Steps>

### Create a New Block

1. Go to the [dashboard overview page](https://app.measurely.dev/).
2. Click **Create block**.

### Configure Block Settings

1. Choose a block type (e.g., "area chart", "combo chart").
2. Configure the **block settings**.

</Steps>

## Block groups

A block group is a collection of blocks that you can use to organize and group related filter categories. You can create block groups to categorize filters based on specific criteria, making it easier to manage and analyze data.

<Callout type="warning">
  Block groups only work with filters. Ensure you have defined filters before
  creating block groups.
</Callout>

<Steps>

### Create a New Block Group

1. Go to the [dashboard overview page](https://app.measurely.dev/).
2. Click **Create block group**.

### Add Blocks to a Group

1. Click on the add new block placeholder.
2. Select the chart you want to add to the group.
3. Configure the block settings.

</Steps>

Blocks are a powerful way to visualize your data and gain deeper insights into your metrics. Use them to create interactive and customizable representations of your metrics and filters.
