# Introduction

Measurely is a simple yet powerful tool that helps you track and analyze key metrics in your projects. By integrating this solution, you can easily collect real-time data and gain valuable insights into your app's performance.

## How Does It Work?

It captures data from your app, enabling you to monitor key performance indicators over time. Designed for easy integration, it helps you focus on what matters most—tracking metrics that impact your app’s success.

### Key Features

- **Real-time Monitoring**: Track your app's performance in real-time.
- **Custom Metrics**: Define custom metrics to track specific events.
- **Data Visualization**: Visualize data with charts and graphs.

## Why Use It?

This tool empowers developers, product teams, and analysts to monitor app performance, uncover trends, and make data-driven decisions. It simplifies the process of collecting and understanding metrics, helping you improve your project with minimal effort.

### Benefits

- **Data-Driven Decisions**: Make informed decisions based on real-time data.
- **Performance Insights**: Monitor app performance and identify bottlenecks.
- **User Engagement**: Understand user behavior and improve user experience.

## FAQ

<details>
  <summary>What is a metric in Measurely?</summary>A metric in Measurely is a
  tracked data point that represents a key value over time. Examples include
  website visits, sales numbers, API requests, and user signups.
</details>

<details>
  <summary>Should I use Measurely for my project?</summary>
  If you need a simple, flexible, and developer-friendly analytics solution,
  Measurely is a great choice. It supports custom metrics, integrations, and
  real-time tracking.
</details>

<details>
  <summary>How do I send data to a metric?</summary>
  You can send data using the Measurely API. Simply make a request to the API
  endpoint with your metric name and value.
</details>

<details>
  <summary>Does Measurely support integrations?</summary>
  Yes! Measurely currently supports **Stripe** for tracking financial data, with
  **Google Analytics** and **AWS CloudWatch** coming soon.
</details>

<details>
  <summary>Is Measurely open source?</summary>
  Yes! Measurely is fully open source. You can contribute or check out the code
  on [GitHub](https://github.com/measurely-dev/measurely).
</details>
