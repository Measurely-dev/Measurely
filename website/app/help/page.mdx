# Help Center

Welcome to the Measurely Help Center! Here, you'll find resources to help you get started, troubleshoot issues, and make the most of our platform. If you can't find what you're looking for, feel free to reach out to us or contribute to our open-source project.

---

## Getting Started

New to Measurely? Check out our [Getting Started Guide](/docs/) to learn how to set up your account, create custom metrics, and use our API.

---

## Frequently Asked Questions (FAQ)

### 1. How do I create an account?

You can create an account by signing up with an external authentication provider like Google or GitHub. Visit our [Sign Up Page](https://app.measurely.dev/register) to get started.

### 2. How do I track custom metrics?

Once you've created an account, you can start tracking custom metrics by navigating to the [new metric page](https://app.measurely.dev/dashboard/new-metric) and following the prompts to add new metrics.

### 3. How do I use the API?

Our API documentation is available [here](/docs/api-references). It provides detailed instructions on how to programmatically interact with your metrics.

## Troubleshooting

### 1. I can't log in to my account.

- Ensure you're using the correct authentication provider (Google or GitHub).
- If you're still having trouble, reset your credentials or contact us at [info@measurely.dev](mailto:info@measurely.dev).

### 2. My metrics aren't updating.

- Check your API integration to ensure it's correctly configured.
- If you're using the web interface, refresh the page or clear your browser cache.

### 3. I’m experiencing payment issues.

- Verify that your payment details are correct and up to date.
- If the issue persists, contact Stripe support or reach out to us at [info@measurely.dev](mailto:info@measurely.dev).

## Open Source Contributions

Measurely is an open-source project, and we welcome contributions from the community! If you'd like to contribute, here's how you can get started:

1. **Visit our GitHub repository**: [measurely-dev/measurely](https://github.com/measurely-dev/measurely)
2. **Create an issue**: If you've found a bug or have a feature request, [create an issue](https://github.com/measurely-dev/measurely/issues/new) on GitHub.
3. **Submit a pull request**: Follow our [contribution guidelines](https://github.com/measurely-dev/measurely/blob/main/CONTRIBUTING.md) to submit a pull request.
