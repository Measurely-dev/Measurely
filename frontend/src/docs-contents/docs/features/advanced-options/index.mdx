## Getting Started with Offset

Measurely's **Offset** feature is part of the advanced options designed to provide deeper insights into your metrics. By using offsets, you can customize the time periods displayed on metric charts to suit your specific analysis needs.

## What is an Offset?

An offset allows you to adjust the starting point of a metric's time frame, making it easier to view data for specific period.

## How to Use the Offset Option

<Stepper>
  <StepperItem title="Step 1: Access Metric Settings">
      Navigate to the **Metrics** section in the Measurely dashboard and select the metric you want to analyze.
   </StepperItem>
  <StepperItem title="Step 2: Open Advanced Options">
      In the metric's dialog, find the **Advanced Options** button and click on it.
   </StepperItem>
  <StepperItem title="Step 3: Choose a Time Frame">
      Select a custom date.
   </StepperItem>  
  <StepperItem title="Step 4: View">
      The metric chart will now display data starting from the selected time frame.
   </StepperItem>
</Stepper>

## Example Use Cases
**Daily Sales Trends:** Use the "Today" offset to monitor sales data in real time.
## 
**Weekly Comparisons:** Apply the "Last Week" offset to evaluate weekly performance trends.
## 
**Custom Period Analysis:** Define a custom start date to focus on a specific promotional campaign or project duration.
